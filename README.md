# Cooly

A utility that helps you deploy Python projects.


## Installation

Install `Cooly` with `pip`:

    $ pip install Cooly

Install development version from `GitHub`:

    $ git clone https://github.com/RussellLuo/cooly.git
    $ cd cooly
    $ python setup.py install
